## Use it

* Add the widget to your *TableView*:

	```xml
	<TableView id="table">
	  <Widget id="is" src="nl.fokkezb.infiniteScroll" onEnd="myLoader" />
	</TableView>
	```
	
	or *ListView*:
	
	```xml
	<ListView id="list">
	  <Widget id="is" src="nl.fokkezb.infiniteScroll" onEnd="myLoader" />
	</ListView>
	```
